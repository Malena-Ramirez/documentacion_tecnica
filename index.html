Introducción

Es impresionante lo que en la actualidad se puede lograr con un sitio web. Puede crear sitios o aplicaciones web que parezcan aplicaciones nativas, pero que se ejecuten en el explorador web. Si le interesa empezar hoy mismo con el desarrollo web, puede encontrarse ante un conjunto abrumador de opciones. En este módulo, se mostrarán los componentes básicos de un sitio web y las herramientas que puede usar para iniciar el desarrollo web. Usará HTML, CSS y JavaScript en conjunto para crear un sitio web sencillo y usará las herramientas de desarrollo del explorador para entender lo que ocurre. Con este conocimiento fundamental, tendrá un contexto mejor para tomar decisiones en el futuro sobre varios aspectos, como elegir marcos de JavaScript para crear el sitio web.

Página web

En una página web, el lenguaje de marcado de hipertexto (HTML) proporciona el contenido, las hojas de estilos en cascada (CSS) definen el aspecto de la página y JavaScript se usa para proporcionar interactividad con el usuario, es decir, su lógica de negocios.

El contenido, el estilo y la lógica de negocios se separan en archivos HTML, CSS y JavaScript, respectivamente. Un principio de diseño en la programación moderna es la separación de los intereses. Dos de los muchos motivos para separar los intereses son la simplicidad y la reutilización. Por ejemplo, al aplicar estilos a los elementos mediante CSS, puede aplicar estilos a todos los elementos de una página, independientemente de la complejidad de la página, en lugar de codificar la apariencia por separado dentro de cada elemento HTML.

HTML

El objetivo es usar el lenguaje de marcado de hipertexto (HTML) para describir la página web que el explorador debe mostrar. ¿No sería buena idea tener una plantilla inicial? De forma cómoda, los editores pueden rellenar automáticamente parte del texto reutilizable o la estructura HTML típica. Ha habido diferentes versiones de HTML. El tipo de documento <!DOCTYPE html> indica que se trata de código de HTML5.

Si bien no vamos a profundizar demasiado en el significado de todos los elementos HTML, sí señalaremos algunos elementos importantes. La etiqueta meta indica información de metadatos que normalmente no será visible para el lector a menos que vea el código fuente en su explorador. Los elementos o etiquetas meta proporcionan información descriptiva sobre la página web. Por ejemplo, ayudan a los motores de búsqueda a mostrar resultados.

El juego de caracteres (charset) UTF-8 puede parecer insignificante, pero es fundamental para establecer cómo se interpretan los caracteres en los equipos. Si falta, charset puede poner en riesgo la seguridad. Detrás de charset hay mucha historia e información técnica. Aquí, lo más importante es que ese texto reutilizable proporciona algunos valores predeterminados significativos.

CSS

Las Hojas de estilos en cascada (CSS) le permiten especificar la apariencia de la página. La idea básica consiste en seleccionar código HTML como destino y, después, definir cuál debe ser el estilo. Por ejemplo, puede seleccionar un cuadro y aplicarle esquinas redondeadas o asignarle un fondo degradado. La CSS se encarga de determinar la apariencia y respuesta de los hipervínculos cuando se interactúa con ellos, aunque la dirección de destino se define en HTML. En CSS3, también puede realizar efectos de animación sofisticados.

Para enlazar un archivo CSS externo desde HTML se debe usar la siguiente etiqueta dentro del head, con el nombre del archivo CSS. 

            <link rel='stylesheet' href='tu_archivo.css'>

Reglas de CSS

Imagine que tiene una llave muy antigua y una serie de puertas a lo largo de un pasillo extenso. En primer lugar, selecciona una puerta y la abre con la llave. Tras entrar en una habitación, puede decorarla como quiera. Puede pintar las paredes de azul o elegir madera para los suelos. Selecciona una habitación y define reglas para el aspecto que debe tener. Puede aplicar el mismo estilo a varias salas, lo mismo que en CSS. Veamos el siguiente ejemplo. 

            
                body {
                    font-family: monospace;
                }
                
                ul {
                    font-family: helvetica;
                }

                li {
                    list-style: circle;
                }
                  
                .list {
                list-style: square;
                }
                
                #msg {
                font-family: monospace;
                }

Puede usar reglas de CSS para aplicar estilo al código HTML. El elemento de lista sin ordenar ul {} es un selector que selecciona el elemento HTML <ul> al que aplicar estilos. La declaración es font-family: helvetica y determina cuál debe ser el estilo. El nombre de la propiedad es font-family y el valor es helvetica. Juntos, la propiedad y el valor constituyen un par clave-valor. Lo que se selecciona es un elemento existente que se ha definido antes en HTML (<body> y <ul>).

Los identificadores y los selectores de clase permiten aplicar estilos a los nombres de atributo personalizados en el código HTML. Un identificador se usa para definir el estilo de un elemento, mientras que las clases pueden aplicar estilos a varios elementos. En el ejemplo anterior, el selector de clases list tiene un prefijo de punto, mientras que el selector de Id. es msg y tiene un prefijo de signo de libra. Los nombres pueden ser arbitrarios, siempre que coincidan con lo que haya definido en el código HTML.

JavaScript

JavaScript (o ECMAScript) es un lenguaje de programación que permite agregar interactividad a la página web. Al hacer clic en un botón, JavaScript es el código que define el evento o el comportamiento que tendrá lugar como, por ejemplo, abrir una ventana emergente. Con JavaScript, puede agregar o quitar contenido de la página web, como texto, sin volver a cargarla. Como desarrollador web, puede usar el explorador para probar y obtener información sobre los scripts.

CSS y JavasCript se pueden agregar directamente a la página HTML, pero es mejor no hacerlo. Para agregar archivos externos de JavaScript se utiliza la etiqueta <script> en <head> o en otro lugar de <body>. Sin embargo, colocar <script> al final de la sección <body> permite que primero se muestre todo el contenido de la página y, después, se cargue el script. 

            <script src='tu_archivo.js'>

Conceptos básicos en JavaScript

  • Variables: Las variables básicamente son contenedores de valores (como números o cadenas de texto). Creas una variable con la palabra clave let (o var) seguida de un nombre para tu variable. Las constantes se utilizan para almacenar valores que no deseas modificar y se crean con la palabra clave const. Puedes asignar un valor a tu variable o constante con un signo igual (=) seguido del valor que deseas darle.
  • Funciones: Las funciones son bloques de código reutilizable que puedes escribir una vez y ejecutar una y otra vez, ahorrando la necesidad de repetir el código todo el tiempo. Son realmente útiles. Hay varias maneras de definir funciones, pero por ahora nos concentraremos en un tipo simple. Aquí hemos definido una función usando la palabra clave function, seguida de un nombre, con paréntesis después de él. Después de eso ponemos dos llaves ({ }). Dentro de las llaves se encuentra todo el código que queremos ejecutar cuando llamamos a la función. Cuando queramos ejecutar el código, escribimos el nombre de la función seguido de los paréntesis.
  • Operadores: Los operadores de JavaScript nos permiten realizar pruebas, hacer cálculos matemáticos, unir cadenas y otras cosas similares.
  • Condicionales: Un bloque de código condicional te permite ejecutar código de manera selectiva, dependiendo de si una determinada condición es verdadera o no. Se parece un poco a una función, pero no lo es. La forma más simple de bloque condicional comienza con la palabra clave if, luego algunos paréntesis, luego unas llaves. Dentro del paréntesis incluimos una prueba. Si la prueba devuelve true, ejecutamos el código dentro de las llaves. Si no, no lo hacemos y pasamos al siguiente segmento del código.
  • Bucles: Los bucles son un muy importante concepto en programación, estos te permiten seguir ejecutando un fragmento de código una y otra vez, hasta que se cumpla una determinada condición.
  • Eventos: Los eventos son cosas que suceden en el navegador — se hace clic en un botón, se carga una página, se reproduce un video, etc. — en respuesta a lo cual podemos ejecutar bloques de código. Las construcciones que escuchan el evento que ocurre se denominan escuchas de eventos, y los bloques de código que se ejecutan en respuesta a la activación del evento se denominan controladores de eventos.

Bonus: Objetos

Un objeto es una colección de propiedades y funciones relacionadas almacenadas en un solo grupo. En JavaScript, un objeto es un entidad independiente con propiedades y tipos. Compáralo con una taza, por ejemplo. Una taza es un objeto con propiedades. Una taza tiene un color, un diseño, un peso, un material del que está hecha, etc. Del mismo modo, los objetos de JavaScript pueden tener propiedades que definan sus características.

Podemos utilizar un iniciador de objetos, que es una lista delimitada por comas de cero o más pares de nombres de propiedad y valores asociados de un objeto, encerrados entre llaves ({}): 

            
                let miObjeto = {
                    nombre: 'taza',
                    color: 'amarillo',
                    material: 'porcelana'
                };
